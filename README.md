# ChatRoom
NodeJs + Socket.io + AngularJs + AngularMaterial 构建的聊天室

**Checkout and Run：**

  > 首先 ,确认你的系统安装了[MongoDB](http://www.mongodb.org/)，和 [Nodejs](http://nodejs.org/ "Nodejs") 。

**1，clone code**

 >  `git clone https://github.com/mz121star/NJBlog.git`


**2，install and run app**

>  `npm start`
